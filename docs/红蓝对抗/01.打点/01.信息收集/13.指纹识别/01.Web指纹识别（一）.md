# 0x05 WAF识别

## 01 手动

一般来说WAF都会拦截敏感文件下载，如`a.sql`、`a.mdb`等，直接添加到网站URL中即可

一些WAF可能不防护这些，那么就可以拿一些XSS、SQLI的payload来测试也可
